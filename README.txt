CHECKSTUDENT

Сервис по распознаванию лиц студентов с помощью face_regognition для автоматического учета посещаемости

Преподаватели много времени и внимания тратят на то, чтобы отметить присутствие студентов. 
Наш проект призван решить эту проблему. 

Мы создаем прототип захватывания лиц, обработки полученных изображений с помощью технологии распознавания лиц
и вывод полученных результатов - фиксировании присутствия человека - в веб-приложении.
